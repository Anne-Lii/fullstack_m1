Ramverk 2 - Bulma

Bulma är ett modernt CSS-ramverk baserat på Flexbox, vilket gör det enkelt att skapa responsiva och stilrena webbplatser. Ramverket är modulärt och har en rad färdiga komponenter och klasser som underlättar webbutveckling utan att behöva skriva mycket egen CSS.
Besök Bulma för mer information.

+ Fördelar

  • Flexbox-baserat, enkelt att skapa responsiva layouter.
  • Lätt att använda, med tydlig dokumentation.
  • Modulärt,man behöver bara ladda de delar som behövs.
  • Mycket tutorials online.

- Nackdelar

  • Saknar inbyggda JavaScript-komponenter.
  • Kan behövas egen CSS för avancerade anpassningar.
  • Fokuset på flexbox kan vara jobbigt om man föredrar Grid.
  • Funktionaliteten är lite begränsad jämfört med vissa andra ramverk.
